2. MongoDB
https://www.mongodb.com/try/download/community

- D: MongoDB/bin
- 在MongoDB下新建db和log文件夹, 还有mongod.cfg
- db =>  D: MongoDB/db
- log => D: MongoDB/log/mongod.log

systemLog:
    destination: file
    path: D: MongoDB/log/mongod.log
    logAppend: true
storage:
    journal:
        enabled: true
    dbPath: D: MongoDB/db
net:
    port: 27017


net start MongoDB
